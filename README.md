## 📁 What You Have (3 Files)

| File | What it is |

| `README.md` | This guide |

## 🚀 Step-by-Step: Get Your Dashboard Live in ~10 Minutes

### STEP 1 — Create a GitHub Account (skip if you have one)
1. Go to **https://github.com**
2. Click **"Sign up"** (top right)
3. Enter your email, create a password, pick a username
4. Verify your email when GitHub sends you a confirmation

---

### STEP 2 — Create a New Repository (your "project folder" on GitHub)
1. Once logged in, click the **"+"** button (top right) → **"New repository"**
2. Give it a name like: `electrawireless-esg`
3. Make sure **"Public"** is selected ✅ *(required for free Streamlit Cloud)*
4. Tick **"Add a README file"** ✅
5. Click **"Create repository"** (green button)

---

### STEP 3 — Upload Your 3 Files
1. You're now inside your new repository — you'll see a file list
2. Click **"Add file"** → **"Upload files"**
3. Drag and drop all 3 files:
   - `app.py`
   - `requirements.txt`
   - `README.md` *(this file — optional but good practice)*
4. Scroll down, click **"Commit changes"** (green button)
5. All 3 files should now appear in your repository ✅

---

### STEP 4 — Create a Streamlit Cloud Account
1. Go to **https://streamlit.io/cloud**
2. Click **"Sign up"**
3. Choose **"Continue with GitHub"** — this links your GitHub account automatically
4. Authorize Streamlit when it asks for permission

---

### STEP 5 — Deploy Your Dashboard
1. On the Streamlit Cloud dashboard, click **"New app"** (top right)
2. Fill in the form:
   - **Repository:** Select `your-username/electrawireless-esg`
   - **Branch:** `main`
   - **Main file path:** `app.py`
3. Click **"Deploy!"**
4. Wait ~2 minutes while it installs everything 🕐
5. Your dashboard will open automatically! 🎉

---

### STEP 6 — Share With Anyone in the World
- Your dashboard now has a **public URL** that looks like:
  `https://your-username-electrawireless-esg-app-xxxxx.streamlit.app`
- Copy this link and share it via email, WhatsApp, LinkedIn — anyone can open it in a browser, no login needed!
- To find the URL anytime: go to **https://share.streamlit.io** and click your app

---

## 🔁 How to Update the Dashboard Later

If you want to change any numbers or text:
1. Go to your GitHub repository
2. Click on `app.py`
3. Click the **pencil icon ✏️** (top right of the file)
4. Make your changes
5. Click **"Commit changes"** (green button)
6. Streamlit Cloud will automatically refresh your live dashboard within ~1 minute!

## ❓ Common Questions

**Q: Does it cost anything?**
A: No! GitHub (public repos) and Streamlit Cloud (1 app) are both free.

**Q: Do viewers need to install anything?**
A: No! They just click your link and it opens in their browser.

**Q: Is my data safe?**
A: All data is built into the app.py file. No external database is used.

**Q: Can I make it private?**
A: Yes — Streamlit Cloud's paid plan supports private apps. For a presentation, the free public link works perfectly.
